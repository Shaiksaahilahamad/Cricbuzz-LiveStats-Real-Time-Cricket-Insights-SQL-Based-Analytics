# 🏏 Cricbuzz LiveStats - Cricket Analytics Platform

A comprehensive cricket analytics platform that combines real-time match data with powerful SQL-based analytics, built with Python and Streamlit.

## 📋 Overview

Cricbuzz LiveStats is a web application that provides cricket enthusiasts, analysts, and learners with valuable insights into the game through:

- Real-time cricket match updates and statistics
- Advanced SQL-based analytical capabilities
- Player search and performance analysis
- Interactive database operations (CRUD functionality)
- User-friendly interface for cricket analytics

## 🚀 Features

### 🏠 Home
- Project overview and technology stack
- Quick navigation to all sections
- Database connection status

### 🏏 Live Matches
- Real-time match scores and updates
- Detailed scorecards with batting and bowling statistics
- Match status and commentary
- Venue information and match details
- Filter by Live, Recent, and Upcoming matches

### 📊 Player Statistics
- Comprehensive player search functionality
- Detailed player profiles and career statistics
- Performance metrics across different formats
- Batting and bowling statistics visualization
- Comparative analysis tools

### 🧮 SQL Analytics
- 25+ predefined analytical queries (Beginner to Advanced)
- Real-time query execution
- Results visualization and export
- Custom SQL query interface
- Sample data generation for testing

### 🛠️ CRUD Operations
- Complete database management interface
- Create, Read, Update, Delete functionality
- User-friendly forms for data manipulation
- Learning tool for SQL operations

## 🛠️ Technology Stack

- **Frontend**: Streamlit
- **Backend**: Python 3.10+
- **Database**: MySQL/PostgreSQL
- **API**: Cricbuzz API via RapidAPI
- **Libraries**: Pandas, Requests, MySQL Connector, python-dotenv
- **Tools**: Git, VS Code, MySQL Workbench
